Data open and editing

How to open project:

  -You can open existing XML file using File->Open

  -You can create new empty project using File->New

How to edit data:

  -You can add point to f(x)table using "Add point" button

        P.S. If X and Y text fields are empty, new point will be (0,0), and (x,y) otherwise

  -You can change values of points by double tap on needed field

  -You can delete point by selecting it in table and tapping on "Delete point" button

  -g(x) can be changed by changing g(x) text field. Function should be inputed using JavaScript syntax

  -To solve task, click "Solve" button. Graph and result table will appear on the screen.
